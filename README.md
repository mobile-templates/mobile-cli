# mobileh5-cli

> mobileh5-cli 是 创建 mobile-h5 产品工程模板的命令行工具

## 安装

```bash
npm config set registry https://repo.huaweicloud.com/repository/npm/
npm install -g mobileh5-cli
```

## 使用

```bash
mobile -v // 查看当前cli版本

mobile -h // 命令帮助信息

mobile create <projectName> // 创建mobile-h5的工程模板，可以指定模板类型及其版本

```
